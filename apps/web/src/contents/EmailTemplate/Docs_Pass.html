1. 면접은 어떤 내용으로 진행되나요?

제출해주신 자료들을 바탕으로 관련된 기술과 경험에 대해 심도 있는 이야기를 나눠요.

2. 직무 면접은 어떻게 진행되나요?

현재 코로나 상황으로 인해 부득이하게 화상으로 직무 면접을 진행하고 있어요.

구글미트로 진행되며 대략 30~50분 정도 소요돼요.

2. 주의사항이 있나요?

Interviewer 와의 원활한 소통을 위해 카메라와 마이크가 필요해요.

시작 이전 카메라 상태와 마이크 상태를 반드시 점검해주시고, 참여시 복장이나 장소등은 Interview 진행을 해치지 않는 선에서 자유에요.
